discoversounds

discoversounds is a tool that helps you discover new music, based on an artist that you already like. It generates 20 suggested artists, along with useful links for the individual artists such as Wikipedia, YouTube and Spotify.

I wrote a blog post on discoversounds here and you can give it a whirl here.

Stack: JavaScript
Deployment: Self-hosted

Statocat

Statocat is a statistics generator for GitHub profiles. It was developed as an entry to UP Singapore's API coLAB Hack Day where it clinched the award for Best Use of Other APIs.

You can give Statocat a whirl here.

Stack: Ruby on Rails, PostgreSQL, JavaScript
Deployment: Heroku

Zero

Zero is a Jekyll theme born out of the need for a clean, developer-friendly and distraction-free reading environment. Zero is built on Bootstrap, making it perfectly responsive on all sorts of devices.
